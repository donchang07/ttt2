# RAG 비용·안전 (Day12 산출물 A-12-O4)

작성일: 2026-06-28 · 추정치(estimate), 실측 토큰 아님

## 비용 추정
- **임베딩(OpenAI text-embedding-3-small, $0.02 / 1M tokens)**: 시드 12문서×1청크×~60토큰 ≈ 720토큰 + 질의 5건×~15토큰 ≈ 75토큰 → 합 ~800토큰 ≈ **$0.00002 미만**.
- **생성(Claude claude-opus-4-8, $5/1M in · $25/1M out)**: 근거 있는 질의 4건, 건당 입력 ~330토큰(시스템+컨텍스트 3청크+질문), 출력 ~50토큰 → 입력 ~1,320 / 출력 ~200 → 약 **$0.012**.
- **합계 1회 평가 ≈ $0.013** (센트 단위). 문서·질의가 늘면 선형 증가.

## 재시도/레이트리밋
- OpenAI·Anthropic SDK 기본 재시도(429·5xx, max_retries=2, 지수 백오프).
- 대량 배치 임베딩 시 429 가능 → 청크 간 지연 또는 배치 크기 축소(현재 1회 호출에 청크 배열 전달로 호출 수 최소화).

## 데이터 전송 범위
- **→ OpenAI**: 임베딩 입력(문서 청크 텍스트, 질의 텍스트)만. 생성에는 OpenAI 미사용.
- **→ Anthropic(Claude)**: 검색된 컨텍스트 청크 + 사용자 질문 + 시스템 프롬프트.
- 둘 다 **더미 문서만** 전송. 실고객 데이터·PII·키 미전송.

## Supabase 저장 범위
- `document_chunks`: 원문 청크 텍스트, metadata(jsonb), 1536차원 벡터, `user_id`(uuid), `document_name`, `chunk_index`.
- 평문 비밀·실 user_id 식별정보·키 미저장(user_id는 uuid 참조).
- RLS로 본인 행만 접근.

## 캡처/제출 안전
- API 키·JWT·토큰 화면 노출 없음(키는 `.env.local` 서버 전용, gitignore).
- 더미 문서만, 실명·연락처·계약번호·내부 가격 없음.
- `seed_eval.mjs`는 비밀값을 출력하지 않음(키 이름·결과만).
