# 개선 우선순위 (Day15 산출물 A-15-O3)

작성일: 2026-06-28 · 영향(impact) × 수정비용(effort) 2×2. 오늘은 **저비용·고영향 1건**만.

## 2×2 매트릭스
```
영향 High │ [오늘 선택] 로그인 후 홈→앱 진입(S)   │ 대용량 PDF 크래시 방지(M, 완료)
          │ 로그인 후 메뉴 노출(S, 완료)          │ 확인메일 온보딩(M, 완료)
─────────┼──────────────────────────────────────┼──────────────────────────────
영향 Low  │ Slack 알림 정보량↑(S, 완료)           │ (없음)
          │ 노트 빈 상태 안내(S, 보류)            │
          └ 저비용(low effort) ───────────────────┴ 고비용(high effort) ─┘
```

## 오늘의 1건 (선택)
- **로그인 사용자가 홈(`/`)에 오면 `/tasks`로 바로 진입** — 발견성(T1·T4)의 잔여분.
  - 영향 **High**(첫 진입 혼란 제거), 비용 **S**(1파일, 서버 가드 추가).
  - 근거: 메뉴는 추가됐지만(직전 사이클), 로그인 사용자가 마케팅 홈에 머물면 앱·관리자 메뉴 자체를 못 봄. 홈에서 앱으로 보내면 메뉴가 바로 보임.
